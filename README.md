# 🧠 Urdu Sentiment Analysis App

This project is a **Streamlit web application** for analyzing sentiment in Urdu text using a fine-tuned transformer model.

---

## 🚀 Features

- Sentence-wise sentiment prediction
- Paragraph analysis
- Confidence scores
- Hugging Face model integration

---

## 🧠 Model

✅ Hosted on Hugging Face:  
https://huggingface.co/abd12-tahir/urdu-sentiment-model  

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
